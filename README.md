## 安装

把这个仓库里的文件放到 Codex 的宠物目录：

- Windows：`C:\Users\<你的用户名>\.codex\pets\yoyo`
- macOS / Linux：`~/.codex/pets/yoyo`

目录里需要有这两个文件：

- `pet.json`
- `spritesheet.webp`

放好后重启 Codex，然后在设置里选择 `yoyo`。
